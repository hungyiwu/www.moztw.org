Firefox 新鮮事

  • 全新功能 在本次更新，Firefox 引進許多改進使影片觀賞更舒適：
    • 子母畫面（Picture-in-Picture）按鈕有新的樣貌與位置，使您更容易的找到並使用。
    • 對於 Mac 使用者，子母畫面現在有了快捷鍵（Option + Command + Shift + 右方向鍵），可以在開始播放前使用。
    • 對於 Windows 使用者，Firefox 現在在硬體解碼影片上使用 DirectComposition，加強了播放期間的 CPU 與 GPU 的運用並改善電池壽命。
  • 全新功能 Firefox 比以往更快了，頁面載入與啟動時間都有效能改善：
    • 基於 flexbox 佈局的網站載入相較過去加快了 20%。
    • 回復瀏覽階段快了 17%，代表您可以更快的找回離開的頁面。
    • 對於 Windows 使用者，開啟新視窗快了 10%。
  • 全新功能 在從 Firefox 工具列儲存網頁到 Pocket 後，您可以瀏覽新的文章。
  • 全新功能 WebRender 持續推出給 更多 Windows 上的 Firefox 使用者。
  • 修正 螢幕閱讀器的段落回報功能現在正確的回報 Firefox 中的段落而非行數。
  • 修正 修補幾種不同的安全弱點。
  • 變更 現在可以根據卡片類型更方便地自動填入信用卡，此外卡片編輯器裡的卡片號碼現在可以被螢幕閱讀器取得。
  • 變更 列印對話框中無效輸入導致的錯誤現在會回報給螢幕閱讀器。
  • 開發功能 MediaSession API 現在預設啟用，這允許網站作者自訂標準媒體撥放互動的行為，提供比以往更多選項。
  • 開發功能 網站工具箱中的網路現在會顯示 server-sent 事件。這允許伺服器在任何時間發送新的資料到網頁上，開發者可以看到過去無法獲取的事件以進行更底層的故障排除。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
