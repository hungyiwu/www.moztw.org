Firefox 新鮮事

  • 全新功能 Firefox 現在可以被設為系統的預設 PDF 閱讀軟體。
  • 全新功能 輔助功能環境工具回報的多層樹狀控制項中的物件名稱不再錯誤的包含更深層的資訊，提供螢幕閱讀器使用者正確層級的內容。
  • 修正 修補幾種不同的安全弱點。
  • 修正 修復一些在螢幕閱讀器上的故障，包含在 JAWS 螢幕閱讀器上經常當掉的問題。
  • 修正 Firefox 網頁工具箱獲得一些重要的修正，使螢幕閱讀器的使用者可以從過去無法使用的工具中獲益。
  • 修正 SVG 元素的 title 與 desc（標籤與描述）現在可以正確地提供給輔助技術產品（例如螢幕閱讀器）。
  • 變更 對設定減少動態效果的使用者，我們降低了像是分頁載入時的動畫數量，為有偏頭痛和癲癇的使用者減少動態效果。
  • 變更 新的附加元件黑名單已經啟用，以提升效能和可擴充性。
  • 企業版 最新版本的 Firefox 加入了一些 Bug 修復與新的政策，您可以在 Firefox 企業版 80 Release Notes 獲得詳情。
  • 企業版 今天是 Firefox 68 ESR (68.12) 最後一次的預定釋出，除非有重要的安全問題在 2020 年 9 月 22 日的 Firefox ERS 78.3 釋出前被發現。Firefox 68 ESR 的使用者將會在 Firefox 78.3 釋出時自動升級到 Firefox 78 ESR 系列。
  • 開發功能 我們在 Firefox Developer Edition 的檢測器加入了 實驗性的側欄面板，幫助開發者利用 MDN 的資料來更快的識別潛在的瀏覽器相容問題。
  • 開發功能 在 網路監視器的請求清單 中，超過等待門檻的慢速請求上會顯示一個烏龜圖示。
  • 開發功能 Firefox 現在支援 RTX 與 Transport-cc 以增強弱網路環境中的通話品質與提供更好的流量估計。這些功能也為許多運用 WebRTC 的網站提供更好的相容性。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
