Firefox 新鮮事

  • 全新功能 我們對更多 使用 Intel 與 AMD GPU 的 Windows 電腦用戶 釋出 WebRender，讓更多使用者都能享受到更高的繪圖效能。
  • 全新功能 在德國的 Firefox 使用者現在會在新分頁看到 Pocket 推薦，精選網路上的奇文妙作。如果您沒看到這些內容的話，您可 參閱此步驟 打開新分頁的 Pocket 文章。
  • 修正 修補幾種不同的安全弱點。
  • 修正 修復一些在螢幕閱讀器上的故障，包含在 JAWS 螢幕閱讀器上經常當掉的問題。
  • 修正 Firefox 網頁工具箱獲得一些重要的修正，使螢幕閱讀器的使用者可以從過去無法使用的工具中獲益。
  • 修正 SVG 元素的 title 與 desc（標籤與描述）現在可以正確地提供給輔助技術產品（例如螢幕閱讀器）。
  • 企業版 最新版本的 Firefox 加入了一些 Bug 修復與新的政策，您可以在 Firefox 企業版 79 Release Notes 獲得詳情。
  • 企業版 密碼政策 的更新使管理員可以強制使用主控密碼。先前的政策可停用主控密碼，但無法強制使用。被要求必須使用主控密碼的使用者，將會在第一次嘗試儲存密碼時，被要求建立主控密碼。
  • 開發功能 新加入的 非同步呼叫堆疊 讓開發者得以追蹤在事件、逾時（Timeout）、Promise 的非同步程式碼。非同步執行鏈會被顯示在除錯器中的呼叫堆疊，以及在主控台的錯誤和網路的發起人裡的追蹤堆疊。
  • 開發功能 帶有錯誤代碼（4xx 或 5xx）的網路回應會在 主控台中作為錯誤 顯示，以便透過相關的紀錄理解問題。請求與回應細節可以被展開和重新傳送來快速除錯。
  • 開發功能 JavaScript 錯誤在主控台外，現在也會在除錯器中顯示。關聯的程式碼會被強調標記並在滑鼠懸停時顯示其錯誤。
  • 開發功能 得力於 Source Map 跨面板處理的優化，現在從檢測器開啟 SCSS 和 CSS-in-JS 會更加可靠。
  • 開發功能 瀏覽器快捷選單（右鍵選單）中的 檢測輔助功能環境屬性 現在預設開放給所有使用者。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
